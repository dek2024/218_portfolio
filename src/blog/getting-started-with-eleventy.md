## Introduction

Eleventy is a simpler static site generator that's incredibly flexible and fast. In this post, we'll explore why Eleventy is a great choice for building static websites.

## Why Eleventy?

- **Simple**: No client-side JavaScript framework required
- **Fast**: Quick build times even for large sites
- **Flexible**: Use any templating language you prefer
- **Zero-config**: Works out of the box with sensible defaults

## Getting Started

First, install Eleventy:

```bash
npm install --save-dev @11ty/eleventy
```

Then create your first template and you're ready to go!

## Conclusion

Eleventy is a powerful tool for building modern static websites. Give it a try for your next project!
