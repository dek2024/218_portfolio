## Overview

This portfolio website showcases my work and blog posts. Built with Eleventy for blazing-fast performance and easy content management.

## Features

- **Static Site Generation**: Lightning-fast page loads
- **Responsive Design**: Works perfectly on all devices
- **Blog Integration**: Share thoughts and tutorials
- **Project Showcase**: Highlight my best work
- **SEO Optimized**: Built-in SEO best practices

## Technical Implementation

The site uses Eleventy's powerful templating system with Nunjucks templates. Content is written in Markdown for easy maintenance.

### Key Technologies

- **Eleventy**: Static site generator
- **Nunjucks**: Templating engine
- **Markdown**: Content format
- **GitHub Pages**: Hosting and deployment

## Challenges & Solutions

One of the main challenges was creating a flexible template system that could handle different content types. I solved this by creating separate layouts for pages, blog posts, and projects.

## Results

The site achieves perfect Lighthouse scores and loads in under a second. The blog and project sections make it easy to showcase my work and share knowledge.
